O que é Open Source Share?

O Open Source Share é um projeto que tem como objetivo a divulgação de projetos open source na comunidade de desenvolvedores, principalmente no Twitter. Inspirado no @ViUmaVaga , o Open Source Share consiste em um bot no twitter que realiza a divulgação de um projeto através de um retweet quando for mencionado. Em desenvolvimento, o projeto é open source, disponível no github, buscando contribuições para melhorias. Mais informações em Como contribuir para o projeto? Obrigado por utilizar o Open Source Share!

Como contribuir para o projeto?

A contribuição com o projeto pode acontecer através do github, fazendo Pull Requests ou Issues. Já em outros ambientes, é possível contribuir através de sugestões na seção Contato do site, no twitter através de DMs (na dm do criador) ou até tweets mencionando o desenvolvedor.

Para contribuir no github, dê um Fork no projeto, clone na sua máquina local, instale as dependências pelo comando pip install -r requirements.txt, crie uma conta de desenvolvedor no Twitter Apps, pegue seus tokens através da conta no twitter para desenvolvedores, defina as variáveis de ambiente com as autenticações e rode o comando python app/app.py

Mais detalhes explicados no README do Projeto.

Sobre o desenvolvedor
